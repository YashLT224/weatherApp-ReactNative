# calculator
## React Native weather mobile App

This project was bootstrapped with. [Expo Cli](https://docs.expo.dev/workflow/expo-cli/).

## Available Scripts

The project is created with-:

```
expo init calculator
```

In the project directory, you can run-:

```
npm start
```
